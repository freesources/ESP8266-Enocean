# ESP8266 Enocean

Enocean für den ESP8266, mittels des EnOcean Pi Funkmoduls.

![Enocean](image.png)

## Verbinden
```
  ESP - ENOCEAN
   TX - RX  10
   RX - TX   8
  Vcc - Vcc  1
  GND - GND  6
```
